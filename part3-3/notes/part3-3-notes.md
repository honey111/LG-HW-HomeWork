#### 什么是渲染

#### 传统的服务端渲染
```js
npm i express // node创建好服务端
使用nodemon index.js // 启动服务器端代码
npm i art-template // 第三方的模板引擎
```
传统的服务端渲染在网页越来越负责的时候，会有很多不足
- 前后端代码完全耦合在一起，不利于开发和维护
- 前端没有足够发挥空间
- 服务端压力大
- 用户体验一般

#### 客户端渲染 
前端更加独立，不局限后端
缺点
- 首屏渲染慢
- 不利于SEO
